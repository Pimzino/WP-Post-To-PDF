Convert blog posts to professionally formatted PDF documents with a customizable download button.

== Description ==

WP Post to PDF adds a customizable PDF download button to your blog posts, allowing readers to save and print your content in a professionally formatted PDF document.

**Features:**

* Customizable download button
* Professional PDF formatting
* Code block preservation
* Image optimization
* Modern admin interface
* Dark mode support
* Responsive design

**Button Customization:**

* Custom text
* Font selection
* Font weight control
* Icon selection
* Placement options

== Installation ==

1. Upload `wp-post-to-pdf` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings â†’ Post to PDF to configure the plugin

== Frequently Asked Questions ==

= Can I customize the button location? =

Yes, you can choose to display the button at the top, bottom, or both locations of your posts. You can also use the shortcode [post_to_pdf] to place it manually.

= Does it support code blocks? =

Yes, the plugin preserves code formatting and syntax highlighting in the generated PDFs.

= Will it work with my theme? =

The plugin is designed to be compatible with any properly coded WordPress theme.
